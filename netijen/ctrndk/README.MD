## SISTEM INFORMASI ALUMNI DENGAN METODE RETRIEVAL SYSTEM

- PHP 7 
- MySQLi

# Cara Penggunaan
1. Buka ```config.php``` di folder ```app/```.
2. Edit HOST, USERNAME, PASSWORD sesuai KONFIGURASI PC/LAPTOP anda.
3. Jalankan lewat Web Server (Lokal/Server Online).
